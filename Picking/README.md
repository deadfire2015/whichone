# MercPicking - 服装选款系统

一个专为服装行业设计的单页面选款应用，支持移动端和PC端，提供表格导入、商品浏览、筛选、购物车式选款和Excel导出功能。

## 功能特性

### 核心功能
- **双表格导入**: 支持尾货表和正价表分别上传
- **实时筛选**: 基于商品分类、童装/成人装、性别、季节等多维度筛选
- **商品浏览**: 支持商品信息查看和筛选
- **Excel导出**: 导出商品清单，包含汇总价格行
- **移动端适配**: 响应式设计，完美适配手机端

### 数据字段支持
- 图片链接、款号SKC、商品编码
- 尺寸、所在仓库、所在仓位、库存数量
- 成本价格、预计售价(人民币/美金)
- 童装/成人装、性别、是否纯棉、季节、商品分类

## 技术栈

- **前端框架**: jQuery 3.7.1
- **UI框架**: PureCSS 3.0.0 (轻量级响应式)
- **表格处理**: SheetJS (xlsx.js)
- **文件格式**: 支持 CSV、Excel (.xlsx, .xls)

## 快速开始

### 1. 直接打开使用
由于是纯前端应用，您可以直接在浏览器中打开 `index.html` 文件使用。

### 2. 本地服务器运行（推荐）
```bash
# 使用Python3
python -m http.server 8000

# 使用Node.js http-server
npx http-server

# 使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`

## 使用指南

### 第一步：数据导入
1. 点击"数据导入"标签页
2. 分别上传尾货表和正价表（支持CSV/Excel格式）
3. 系统会自动验证数据格式并加载

### 第二步：浏览选款
1. 在"浏览选款"页面查看所有商品
2. 使用筛选条件快速定位目标商品
3. 点击商品卡片查看详细信息

### 第三步：浏览商品
1. 查看商品详细信息
2. 使用筛选功能快速定位目标商品

### 第四步：导出Excel
1. 点击"导出Excel"按钮下载商品清单

## 常见问题

### Q: 为什么上传文件后没有显示商品？
A: 请检查您的表格文件是否包含所有必需字段，且格式正确。

### Q: 移动端使用体验如何？
A: 应用已针对移动端优化，支持触摸操作和响应式布局。

### Q: 数据安全吗？
A: 所有数据处理都在浏览器本地完成，数据不会上传到服务器。

### Q: 支持批量操作吗？
A: 支持批量添加到心愿清单功能。
